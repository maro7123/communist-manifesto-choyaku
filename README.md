# 共産党宣言 現代語超訳プロジェクト

マルクス／エンゲルス『共産党宣言』（1848）を、**現代を生きるわたしたちのための読み物**として再構成するプロジェクトです。

**公開版：** https://maro7123.github.io/communist-manifesto-choyaku/

---

## 全体計画（3ステップ）

いきなりラノベを書くのは難しかったので、3ステップに分けました。

### ステップ①：現代インテリ注釈版（**進行中・約26%**）

青空文庫版（堺利彦 1921 改訳）とドイツ語原典（marxists.org）、英訳（Project Gutenberg の Moore 1888）をベースに、**モダンな言葉**に置き換えたり、注釈を加えたりしながら訳しなおします。

- 原典の論理構造は崩さない
- 専門用語の定訳は保つ
- 難読漢字は言い換え or ひらがなに開く
- 現代の労働者・その予備軍が読めるトーンに
- 現代的な例（AI脅威論、Uber配達、ラッダイト、日本車打ち壊しなど）を注釈で補う

### ステップ②：要点とそうでないところを分ける

①をベースに、論理の骨格と枝葉を整理して、ラノベ化のための「下敷き」を作ります。

### ステップ③：ラノベ化（まだ気持ちが続いていたら）

プロット・キャラ設計・ノベル本文。タイトル案は『気がついたら工場でコキ使われていた件』。

### 通底するテーマ：ホラー三部作

訳語・表現は以下の連鎖を保つように選んでいます。マルクス自身が仕込んだ「ブルジョアジーの自己破滅」という物語構造を、日本語版でも一貫した意匠で伝えます。

- **序文の「妖怪」**：ブルジョアジーが**外から**見る恐怖（得体不明だが迫ってくる）
- **I章の「魔法使い」**：自分が召喚した地底の魔物を制御できなくなった魔法使い
- **I章結末の「墓掘り人」**：ブルジョアジーが**自ら生み出した**死の使者（プロレタリアート）

## ビルド・デプロイ

`md` を正本、`html` は `pandoc` で自動生成する設計です。

| コマンド | 用途 |
|---|---|
| `make html` | `choyaku.md` → `choyaku.html` と `index.html` を生成 |
| `make deploy` | master → gh-pages マージして push（サイト更新） |
| `make preview` | ブラウザでローカル確認 |
| `make status` | GitHub Pages のデプロイ状態確認 |
| `make public` | リポジトリを public に切り替え（Pages 公開） |
| `make private` | リポジトリを private に切り替え |
